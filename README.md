# Relogio-Digital-Iniciante-
# Relógio Digital com JavaScript  Projeto simples e funcional de relógio digital, desenvolvido do zero com HTML, CSS e JavaScript, conforme o tutorial do canal Bigode Ensina.
---

## ✨ Funcionalidades
✅ Exibe horas, minutos e segundos
✅ Formata com zero à esquerda (ex: 05 em vez de 5)
✅ Atualiza automaticamente a cada segundo
✅ Layout escuro igual ao modelo do vídeo
✅ Responsivo para celulares e telas pequenas

---

## 📁 Estrutura do Projeto

relogio-digital/
├── index.html          # Estrutura da página
├── css/
│   └── style.css       # Estilos e aparência
└── js/
└── script.js       # Lógica e atualização do horário
---

## 🛠️ Tecnologias Utilizadas
- *HTML5* — estrutura dos elementos
- *CSS3* — estilização e alinhamento
- *JavaScript* — objeto Date e setInterval para atualização

---

## ▶️ Como Executar
1. Salve todos os arquivos mantendo a estrutura de pastas
2. Abra a pasta no VS Code
3. Use a extensão *Live Server* ou abra diretamente o arquivo index.html no navegador
4. O relógio já começa funcionando automaticamente
